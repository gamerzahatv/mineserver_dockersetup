Potainer เป็นเครื่องมือสำหรับจัดการและควบคุมระบบ Docker ที่ใช้งานได้ง่ายและมีอินเตอร์เฟซกราฟิกผู้ใช้ (GUI)

Filestash เป็นแพลตฟอร์มที่ออกแบบมาเพื่อการจัดการและแชร์ไฟล์ออนไลน์อย่างรวดเร็วผ่านทางเว็บเบราว์เซอร์

JupyterHub เป็นแพลตฟอร์มสำหรับการจัดการและให้บริการ Jupyter Notebook แบบหลายผู้ใช้พร้อมกันผ่านทางเว็บบราวเซอร์
